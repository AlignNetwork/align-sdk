## Usage

To use the SDK, ensure that your root component is wrapped with the `AlignProvider`.

### Example

```tsx
// src/index.tsx or src/App.tsx
import React from "react";
import ReactDOM from "react-dom";
import App from "./App";

const App = () => (
  <AlignProvider>
    <MyComponent />
  </AlignProvider>
);
```

Align Default Interaction Types:

1. `file`
2. `nft`
3. `meme`
